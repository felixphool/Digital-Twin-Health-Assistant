This isn't just another health app – it's a **revolutionary AI-powered platform** that:

- 🧠 **Intelligently processes** medical PDFs and lab reports using advanced OCR and AI

- 💬 **Chats with you** about your health using medical-grade AI agents
- 🎯 **Provides personalized insights** and actionable health recommendations
- 📈 **Tracks your progress** with beautiful visualizations and trend analysis
- 🚨 **Alerts you** to potential health concerns that need attention

Think of it as having a **24/7 medical consultant** in your pocket that never gets tired of explaining things and always has your best health interests at heart.

## 🚀 Quick Start

### 1. **Clone the Repository**
```bash
git clone https://github.com/yourusername/team-5.git
cd team-5
```

### 2. **Backend Setup (Python)**
```bash
cd backend

# Create virtual environment
python -m venv venv

# Activate it (Windows)
venv\Scripts\activate
# OR (Mac/Linux)
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Start the server
uvicorn app.main:app --reload
```

### 3. **Frontend Setup (React)**
```bash
cd frontend

# Install dependencies
npm install

# Start development server
npm run dev
```

### 4. **Open Your Browser**
Navigate to `http://localhost:5173` and start exploring!

## 🐛 Troubleshooting

### **Common Issues**

**"Python not found"**
- Make sure Python is installed and added to PATH
- Try `python3` instead of `python`

**"Node modules not found"**
- Run `npm install` in the frontend directory
- Clear npm cache: `npm cache clean --force`

**"Port already in use"**
- Change ports in the startup commands
- Kill existing processes: `lsof -ti:8000 | xargs kill`

**"OCR not working"**
- Install Tesseract: `brew install tesseract` (Mac) or `apt-get install tesseract-ocr` (Linux)
- Windows: Download from [GitHub releases](https://github.com/UB-Mannheim/tesseract/wiki)
